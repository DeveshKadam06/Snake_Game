# 🐍 Snake Game using Python Turtle

A classic Snake Game built using Python's Turtle graphics library.

## Features

- 🐍 Smooth snake movement
- 🍎 Random food generation
- 📈 Live score tracking
- 💀 Collision detection
- 🔄 Press **Y** to Play Again
- ❌ Press **N** to Exit

## Installation

Clone the repository

```bash
git clone https://github.com/DeveshKadam06/Snake_Game.git
```

Go into the folder

```bash
cd Snake_Game
```

Run

```bash
python main.py
```

---

## Controls

| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
| Y | Play Again |
| N | Exit |
